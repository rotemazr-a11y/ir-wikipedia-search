# Quick Start - Wikipedia Search Engine (macOS)

**TL;DR: 7 commands to get your search engine running on MacBook Air**

---

## Prerequisites (macOS)

```bash
# Install dependencies (if not done)
# Note: Use python3 explicitly on macOS
python3 -m pip install --user flask nltk numpy requests

# Download NLTK stopwords
python3 -c "import nltk; nltk.download('stopwords')"
```

**If you get SSL errors on macOS:**
```bash
# Fix SSL certificate issues (common on macOS)
/Applications/Python\ 3.*/Install\ Certificates.command

# Or install via Homebrew (recommended)
brew install python3
python3 -m pip install flask nltk numpy requests
```

## Common Commands (macOS)

**Test specific endpoint:**
```bash
# Basic search
curl -s "http://localhost:8080/search_body?query=test" | python3 -m json.tool

# Title search
curl -s "http://localhost:8080/search_title?query=test" | python3 -m json.tool

# Anchor search
curl -s "http://localhost:8080/search_anchor?query=test" | python3 -m json.tool

# PageRank lookup (POST request)
curl -s -X POST http://localhost:8080/get_pagerank \
     -H "Content-Type: application/json" \
     -d '[1,2,3]' | python3 -m json.tool
```

**Evaluate specific endpoint:**
```bash
python3 scripts/evaluate_map.py --endpoint search_body
python3 scripts/evaluate_map.py --endpoint search_title
```

**Debug mode:**
```bash
# See scores in Flask console
python3 scripts/evaluate_map.py --quiet  # Summary only

# Verbose mode (shows HTTP headers)
curl -v "http://localhost:8080/search_body?query=test"

# Pretty-print JSON (macOS built-in jq alternative)
curl -s "http://localhost:8080/search_body?query=test" | python3 -m json.tool | less
```

**macOS-specific tips:**
```bash
# Open new Terminal tab (instead of window)
# Press: ⌘+T

# Stop Flask server gracefully
# Press: Ctrl+C (not ⌘+C)

# Check if port 8080 is in use
lsof -i :8080

# Kill process on port 8080 if needed
kill -9 $(lsof -ti :8080)
```

## Troubleshooting

**Problem: Empty results**
→ Check stemming: Indices must use `stem=False`

**Problem: MAP@10 < 0.1**
→ Run unit tests to verify formulas

**Problem: Server won't start**
→ Check `INDEX_DIR` points to correct directory

**Problem: Can't build indices**
→ See [DETAILED_SETUP_GUIDE.md](DETAILED_SETUP_GUIDE.md) Section 2
